# skills/

**Только Claude Code Skills** — пакеты `SKILL.md`, которые ставятся в `~/.claude/skills/`.

Признак: устанавливается через `git clone … ~/.claude/skills/`, `/plugin install`, или явно описан как «skill для Claude Code». Всё остальное инструментальное → `tools/`.

Файлы — kebab-case `.md` с YAML frontmatter и ссылкой на репозиторий скилла.
