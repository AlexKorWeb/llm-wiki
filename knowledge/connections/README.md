# connections/

Связи между темами: «X vs Y», «X + Y», экосистемы, рабочие процессы, паттерны сцепления нескольких концептов/инструментов.

Здесь живут страницы, которые не описывают одну сущность, а **соединяют** несколько — карты, сравнения, воркфлоу.

Файлы — kebab-case `.md` с YAML frontmatter и викилинками на связываемые страницы.
